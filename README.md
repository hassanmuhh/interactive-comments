## Technologies used : 
- React
- HTML
- CSS (Pure NO Frameworks)

## Features in Project :
- Adding , Editing and Deleting your Comments or Replies.
- Reply to any comment.
- Comments Sorted by date.
- Upvoting Comments.
- Updating Comment Content (SOON).
